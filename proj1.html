Learning Objectives

Overall objective: help you understand what BLITZ is and how to use it.

Specific objectives:

  • Read the Blitz documentation.
  • Get familiar with the Blitz CPU's instruction set and assembly language.
  • Get familiar with the KPL programming language syntax and semantics.
  • Download and install the Blitz system.
  • Learn how to run, understand, and debug simple Blitz programs.
